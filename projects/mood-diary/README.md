# Mood Diary Portfolio Site

这是一个静态作品集网站，可以直接部署到 GitHub Pages、Netlify 或 Vercel。

## 文件结构

- `index.html`：网站主页面
- `assets/`：App 截图资源

## 本地预览

直接双击 `index.html` 即可打开。

## 部署建议

### Netlify
1. 登录 Netlify
2. Add new site
3. 选择 Deploy manually
4. 拖拽整个 `mood_diary_portfolio_site` 文件夹上传

### GitHub Pages
1. 新建 GitHub 仓库
2. 上传 `index.html` 和 `assets/`
3. Settings → Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`
